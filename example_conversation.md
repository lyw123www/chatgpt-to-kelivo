### 1) MEXC Kickstarter（投票空投）—— 小资金性价比高

**规则要点（官方）**

• 资格：**至少空闲1笔合约交易 + 持有 ≥ 5 MX 连续24小时**；活动期内用MX"承诺"投票，按票数效率瓜分奖池；投票结束后MX"解锁"，本金可取。

• 成本：主要是**MX的短期价格波动风险** + 少量合约手续费（开平仓各0.02%）；不需要大量交易，不需要长期锁仓。

• 收益：取决于**当期项目热度、参与人数、你的MX持仓量**；历史上小额参与（几十到几百MX）也能拿到几U到几十U的空投，ROI可观但不保证。

**一句话建议**

如果你愿意短期持有MX（或已经有），且能接受MX价格小幅波动，这是**门槛最低、适合小白试水**的方式；记得在活动开始前24小时准备好MX，活动结束后及时解锁。

**操作步骤**

1. 注册/登录 MEXC，完成KYC
2. 购买 ≥ 5 MX（建议多买一些以提高投票权重，如 50-100 MX）
3. 在"Kickstarter"页面找到当前活动，点击"投票"
4. 输入要投的MX数量，确认锁定
5. 等待活动结束，查看奖励并解锁MX

当然可以！下面是一个简单的 Python 脚本来计算投资回报率（ROI）：

```python
def calculate_roi(investment, return_value):
    """
    计算投资回报率
    
    参数:
        investment: 初始投资金额（U）
        return_value: 最终回报金额（U）
    
    返回:
        ROI 百分比字符串
    """
    if investment <= 0:
        return "投资金额必须大于 0"
    
    roi = (return_value - investment) / investment * 100
    return f"ROI: {roi:.2f}%"

# 示例 1：投资 150U，回报 200U
result1 = calculate_roi(150, 200)
print(result1)  # 输出: ROI: 33.33%

# 示例 2：投资 300U，回报 250U（亏损）
result2 = calculate_roi(300, 250)
print(result2)  # 输出: ROI: -16.67%

# 示例 3：投资 100U，回报 150U
result3 = calculate_roi(100, 150)
print(result3)  # 输出: ROI: 50.00%
```

### 使用说明

1. 将 `investment` 替换为你的实际投资金额
2. 将 `return_value` 替换为你的最终回报金额
3. 运行脚本即可看到 ROI 百分比
